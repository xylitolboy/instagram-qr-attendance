How it works

1

Run the setup wizard

Paste the backend script into Google Apps Script, deploy it as a Web App, and paste the URL into the setup wizard. Takes about 5 minutes.

2

Open the admin panel before each event

Enter the session number and date, then click Generate. Download or print the QR code PNG.

3

Display the QR code at your event

Attendees scan it with their phone camera, enter their Instagram handle, and tap Submit.

4

Your Google Sheet updates in real time

New guests are added as rows. Returning guests get their count incremented and the session date marked with O.
